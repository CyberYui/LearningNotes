# Linux 系统中安装 Tomcat 解压版 ( 副 Java 安装 )

--------------

* 所以我们需要首先创建两个相关目录 , 我的习惯是把这些工具放到 home 目录中的同名目录下

* 也就是如果要安装 Java , 那我会先在 home 目录中创建 Java 目录 , 然后在里面保存我的 Java 安装包
